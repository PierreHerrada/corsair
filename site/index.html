Slack-First Workflow

Tag @Corsair in any channel. The bot extracts title, description, and acceptance criteria using Claude, then creates a task automatically.

Three-Stage Agent Pipeline

Plan, Work, Review — each stage manually triggered from the web UI. Humans stay in the loop while agents do the heavy lifting.

Kanban Board

Visual task management with drag-and-drop columns. Track tasks from Backlog through Planning, Working, Reviewing, to Done.

Real-Time Log Streaming

Watch Claude Code execute live via WebSocket. See the agent's plan, code changes, and tool usage as they happen in real-time.

Cost Tracking

Per-task and per-stage token usage and cost breakdown. Know exactly what every AI operation costs with full input/output token accounting.

Plugin Architecture

Extensible integration system. Slack, Jira, and GitHub ship built-in. Add Linear, GitLab, or your own integrations without touching core code.

Get Started in 60 Seconds

Clone, configure, run.

terminal 
# Clone the repository
git clone https://github.com/PierreHerrada/clipper-ai.git
cd clipper-ai

# Configure environment
cp .env.example .env
# Edit .env with your Slack, Jira, GitHub tokens

# Start everything
docker-compose up --build

# Open http://localhost in your browser
